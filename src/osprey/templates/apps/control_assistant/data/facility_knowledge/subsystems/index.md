# Subsystem

* [Experimental Stations (ES)](/subsystems/experimental-stations.md) - Six end-stations where users conduct experiments, each with independent motion control, detectors, and sample environments.
* [Personnel Safety System (PSS)](/subsystems/pss.md) - Access control and radiation interlock system governing beam permits, door status, and area searches.
* [Primary Source (PS)](/subsystems/primary-source.md) - Main particle/photon source system including power supplies, vacuum, RF, and diagnostics subsystems.
* [Timing System (TIM)](/subsystems/timing.md) - Facility-wide synchronization and triggering system providing event-based timing to all subsystems.
* [Transport and Delivery (TD)](/subsystems/transport-delivery.md) - Beam transport, switching, and delivery to experimental stations including magnets, steering, and interlock systems.
* [Vacuum System (VAC)](/subsystems/vacuum.md) - Ultra-high vacuum across all beamline sections, maintained by ion pumps, turbomolecular pumps, and gate valves.
